# MortgageLoanPlan

Spring Boot + Thymeleaf で作った住宅ローン返済シミュレーターです。  
画面で入力した値をもとに、月次の現金・ローン残高・純資産を計算し、折れ線グラフとログテーブルで表示します。

## 入力パラメータ

- 現金初期値
- 住宅ローン残高初期値
- 年間貯金増加
- ローン金利
- 投資利回り
- 毎月のローン返済額
- シミュレーション年数

## 起動方法

```bash
mvn spring-boot:run
```

起動後、`http://localhost:8080` にアクセスしてください。

## 仕様（現在の計算ロジック）

- 年間貯金増加を12で割って、毎月の現金増加として加算
- ローンに月利を加算してから毎月返済額を控除
- 月次処理の最後に投資利回り（月利）を現金へ反映
- 各月のログ（現金・ローン残高・純資産）を表示
