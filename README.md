
住宅ローンの返済計画をシミュレーションします。

毎月の収支・投資・住宅ローン残高をシミュレーションしながら、  
「一定期間後にローン残高がどうなっているか」を確認できます。

---

## 概要

以下のパラメータを設定し、  
**数年〜数十年後に住宅ローンがどの程度減っているか** を確認できます。

- 現在の手元資金（現金）
- 住宅ローン残高
- ローン金利
- 毎月の返済
- 初月にまとめて返済する金額（一括返済）
- シミュレーション年数

以下のような検討に役立つことを目的としています。
- 今ある資金をどう使うべきか？
  - すぐに繰上返済するか
  - 少しずつ返すか
  - 毎月どの程度返すのが現実的か
